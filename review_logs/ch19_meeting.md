## Minor 이슈 처리 결정

편집장 결정: Minor 3건 및 강사 제안 1건은 모두 **HTML 내 코드 주석 보완으로 처리 완료**

### 처리 내역

**M01 (HAL_I2C Receive 주소)**
처리 방법: 코드 예제에 `/* HAL이 R/W 비트를 자동으로 설정 — addr << 1 만 전달 */` 주석 추가
→ 코드 동작에 영향 없으며 교육적 명확성 확보

**M02 (CMSIS 함수 주석)**
처리 방법: `__get_PRIMASK()` 사용 코드에 `/* CMSIS 코어 함수 — cmsis_gcc.h */` 주석 포함
→ 교재 본문에 "CMSIS" 용어가 이미 등장하므로 충분

**M03 (스택 심볼)**
처리 방법: 코드 주석으로 "실제 프로젝트에서는 _Min_Stack_Size 링커 심볼 활용 권장" 설명 추가
→ 교육 예시 수준으로 충분

**강사 제안 FAQ (5절)**
처리 방법: 5절에 FAQ aside 추가 ("Mock 테스트가 통과하면 실물 보장되나요?")

5. **누적 성장 완성**: v0.1~v2.2 전체 여정이 8절 회고에서 통합 정리
6. **산출물 완비**: HTML 원고, SVG 4개, C 코드 3개, 리뷰 로그 5개 모두 생성
