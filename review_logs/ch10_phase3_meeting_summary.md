## 3. 우선순위별 필수 수정사항

### 필수 수정 (Phase 4에서 처리)

#### 🔴 **Critical 필수 1**: 기어비 설명 단계화

**문제점**: 공식 "1.8° × (1÷64) × 16 = 5.625°"가 추상적
**영향**: 초급자 불안 유발 + 강의 어려움
**수정 위치**: manuscript §1, line 160-171

**권장 수정안**:
```html
<h3>스트라이드 계산: 기어비의 마법</h3>

<p>28BYJ-48은 단순한 모터가 아닙니다.
내부에 정밀한 기어 장치(gearbox)가 포함되어 있습니다.</p>

<p>스텝 모터의 스트라이드는 다음과 같이 계산됩니다:

<strong>Step 1: 기본 스텝 각도</strong>
일반적인 스텝 모터의 기본 스텝은 1.8°입니다.

<strong>Step 2: 내부 기어 감속</strong>
28BYJ-48은 내부에 64:1 감속 기어가 있습니다.
64:1은 "입력이 64번 회전할 때 출력은 1번 회전"한다는 의미입니다.

<strong>Step 3: 풀 스텝 시퀀스</strong>
4개 코일을 순차적으로 제어하므로, 실제 스트라이드는:
360° ÷ 64 스텝 = 5.625°

결론: 360° 회전하려면 정확히 64개 스텝이 필요합니다.
```

**수정 시간**: 20분 (html 편집)

#### 🔴 **Critical 필수 2**: 여자 시퀀스 배열의 명확한 설명

**문제점**: "왜 A → A+B → B → B+C 순서인가?" 설명 부족
**영향**: 코드 이해도 저하, 강의 중 질문 예상
**수정 위치**: manuscript §2, line 221-231 또는 §3 코드 주석

**권장 수정안**:
```html
<h3>여자 시퀀스 순서의 의미</h3>

<p>왜 A → A+B → B → B+C 순서일까요?
이 순서는 모터 축의 자기장을 <strong>시계방향으로 순차적으로 회전</strong>시키기 위함입니다.</p>

<p>각 단계에서 활성화된 코일은 자기장을 만들고,
이 자기장의 방향이 순차적으로 변하면서 모터 축이 한 스텝씩 회전합니다.</p>

<p>역방향(반시계방향)으로 회전시키려면 역순(B+C → B → A+B → A)을 사용합니다.
코드의 stepper_step_backward() 함수를 보면,
motor_step_index = (motor_step_index + 3) % 4;
로 표현되어 있으며, 이는 배열을 역순으로 순회하는 것과 같습니다.</p>
```

**수정 시간**: 15분 (html 추가)
