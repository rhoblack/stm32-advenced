## 감정 곡선 반영 확인

| 절 | 설계된 감정 | 구현 장치 |
|----|-----------|----------|
| §3.1 | 위기감 + 호기심 | 3분 공포 체험, 스파게티 코드 스크롤, "야근 방지 기술" |
| §3.2 | 안심 + 체계적 이해 | 레스토랑 비유, 4계층 정의, 인터페이스 미니 실습 |
| §3.3 | 참여감 + 성취감 | 워크시트 분류, 위반 발견 ("이건 고쳐야 하는구나") |
| §3.4 | 약간의 불안 → 아하 | "FSM이 어려운데..." → "이미 하고 있었다!" |
| §3.5 | 창조 성취감 + 기대감 | 설계도 직접 작성, 회색 블록 → "앞으로 채워나간다" |
| 정리 | 기대감 극대화 | "다음에 첫 블록이 색칠된다" |

## 시퀀스 다이어그램 관련 참고사항

§3.5의 시퀀스 다이어그램은 SVG 대신 `<pre><code>` ASCII 형태로 포함했습니다.
이유: 시퀀스 다이어그램을 "맛보기" 수준으로 축소했으므로 별도 SVG 파일을 만드는 것은 과도합니다.
Ch04부터 SVG 시퀀스 다이어그램을 본격 사용합니다.
