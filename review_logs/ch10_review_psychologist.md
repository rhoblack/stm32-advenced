#### Stage 3: 불안 유발 구간 (§1 기어비)

**교재 내용**:
```
기본 스텝 = 1.8°
기어비 = 64:1
실제 스트라이드 = 1.8° × (1 ÷ 64) × 16 = 5.625°
```

**문제점** (심리학적):
🔴 **불안 수준이 과도함**

- 독자는 갑자기 "1.8° × (1÷64) × 16" 공식을 마주함
- "왜 16을 곱하나?" 의문 발생 (공식 설명 부족)
- 기어비 개념이 추상적 (명확한 단계 설명 없음)
- **결과**: 불안이 공포로 전환될 수 있음

**해결책** (권장):

§1 뒤에 **미니 안심 섹션** 추가:

```html
<aside class="tip">
  💡 <strong>기어비를 직관적으로 이해하기</strong><br>
  자전거를 생각해보세요. 앞 페달(입력)이 64번 회전하면,
  뒷 바퀴(출력)는 1번만 회전합니다. 이것이 64:1 기어비입니다.
  <br><br>
  스텝 모터도 같습니다:
  <ul>
    <li>모터 내부 코일: 64번 자기장 회전</li>
    <li>출력축 회전: 1번</li>
    <li>결과: 1.8° × (1/64) = 0.028° (한 번 회전할 때)</li>
    <li>풀스텝은 4가지 코일 상태: 0.028° × 200 = 5.625° (최종 스트라이드)</li>
  </ul>
</aside>
```
